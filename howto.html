WEBCRAWLER TOOL

EURAKNOS webcrawler tool main page

The main page of the webcrawler tool consists of three parts. There are six tiles directly under the menu functions. The list of the individual webcrawlers is located under these tiles.

Each tile shows the total and the distinct number of related contents and their metadata availability ratio.

  • TEXT tile shows the number of harvested contents that are in HTML, PDF, DOCX, XLSX, EPUB file formats.
  • PRESENTATION tile shows the number of harvested contents in PPTX and Prezi format.
  • IMAGE tile shows the number of harvested contents in JPG, PNG, BMP, SWF, SVG formats.
  • VIDEO tile shows the number of harvested contents in YouTube video, Vimeo video, MP4 formats.
  • AUDIO tile shows the number of harvested contents in Spotify format.
  • OTHER tile shows the number of harvested contents in Google Map, ZIP and Mailchimp formats.

EURAKNOS webcrawler tool functions

Create and start an individual webcrawler

Using this function you will be able to create, configure and start your own webcrawler.

After clicking the “Add crawler” button on the main page, you must set the name and the starting URL of your webcrawler. Optionally you can set or add extra parameters that influence the working of your webcrawler. By clicking the “Start” button your webcrawler starts.

Adjustable parameters:

Parameters Description Mandatory (Y/N) Default value
Starting URL The starting URL of the web crawler. Y Empty
Base URL The base URL defines the constant prefix of contents’ URLs. In some special cases the starting and the base URL differs. N Empty
Crawler’s name The name of the web crawler. Mandatory field. Y Empty
External content download If set, crawler will download contents that are linked from the starting site but are not hosted on it. N On
META Title Harvesting meta title from the site. N Off
META Keywords Harvesting meta keywords from the site. N Off
META Description Harvesting meta description from the site. N Off
Heading 1 Harvesting level 1 heading (H1) tags from the site. N Off
Heading 2 Harvesting level 2 heading (H2) tags from the site. N Off
Heading 3 Harvesting level 3 heading (H3) tags from the site. N Off
Other searching elements In this parameter group it is possible to add some extra searching options to refine the crawling process. The first field contains the DIV class name, the second one is its the reference name in the database. N Empty, Empty

Notice: Use this function with responsibility! Crawling and scraping internet sites are not always a warmly welcomed activity.

Details and Export details

Applying this feature it is possible to view the terms and results of any individual webcrawler searches. The appearing chart shows the type, proportion, and number of files downloaded, as well as the percentage of metadata extracted.

By clicking on the “Export details” button, the download statistics of the current webcrawler can be exported in Excel format.

Notice: In case at least one metadata value can be retrieved automatically for a specific content, we calculate that metadata is available for that content. We do not take into account whether the metadata is insufficient or incorrect. This indicator will be refined soon.

Export metadata

Using this feature, you can export all automatically extracted metadata of an individual webcrawler, in Excel format, regardless of whether the crawling process is still running or has been ended.

Clicking the “Export metadata” button will start the export process.

Stop running individual webcrawler

You can use this feature to stop the selected running individual webcrawler. The process completes the search and cannot be restarted afterwards.

Clicking the “Stop” button will stop the search.

Delete completed individual webcrawler and its data

You can use this feature to delete data and metadata databases generated as a result of a completed individual webcrawler process. Deletion is permanent and cannot be undone.

Click the “Delete” button to delete the individual webcrawler and its downloaded data and metadata databases.

Export all download statistics

Using this feature, you can export the download statistics of all individual webcrawlers, in Excel format, regardless of whether the crawling process is still running or has been ended.

Clicking the “Export all” button will start the export process.
